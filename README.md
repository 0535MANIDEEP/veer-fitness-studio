# Veer Fitness Studio

Official website for Veer Fitness Studio, a premier gym in Shankarpally, Hyderabad.
Built to showcase the gym's facilities, pricing, and expert training led by Rajesh Goud Tatikonda.

## Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: Vanilla CSS Modules (Variables & Responsive Design)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Font**: Inter & Oswald (via `next/font`)

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/veer-fitness-studio.git
   cd veer-fitness-studio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Deployment

This project is optimized for deployment on [Vercel](https://vercel.com).

1. Push code to GitHub.
2. Import project into Vercel.
3. Deploy.

Live URL: [https://veer-fitness-studio-shankarpally.vercel.app/]
